# 💫 About Me:
I’m Arman Phaugat, a Computer Science undergraduate at Manipal University Jaipur with a strong focus on backend engineering, system design, and scalable web architectures. I enjoy building high-performance, production-ready systems that handle concurrency, security, and real-time data efficiently.<br><br>I have hands-on experience developing robust backend services using Node.js, Express, MySQL, MongoDB, Redis, and Docker, with a strong emphasis on ACID transactions, authentication & authorization, caching, rate limiting, and job queues.
